# SEEDi

Sustainable Evidence for Emerging Decisions

## Decision Intelligence Platform

This repository contains a code bundle for the Decision Intelligence Platform. The original design is available at https://www.figma.com/design/4X6bIalBVPPkGFwc37Xlna/Decision-Intelligence-Platform.

## Running the code

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Contributions and issues are welcome.
